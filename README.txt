#README

Le programme qui suit permet de calculer 
et de récupérer dans un fichier l'ensemble des 
intéaractions existantes à l'intérieur d'une protéine.

LANCEMENT DU PROGRAMME:
Dans un terminal, se mettre dans le répertoire:
~/Yoann_Missolo_Projetpy/Programme

Dans un terminal, entrer la commande suivante:
./Programme.py [Nom_du_fichier_.pdb] >[Nom_du_fichier_de_sortie]

EXEMPLE:
./ProjetPy.py 1zni.pdb >filout.txt

Voir un exemple de fichier de sortie dans le dossier Résultat.
